# String Operations

A Java program designed to perform essential string manipulation techniques using methods, loops, and character tracking.

## 🚀 Features
* **Palindrome Checker:** Validates whether a given string reads the same backward as forward.
* **Vowel Counter:** Counts the total number of vowels present in a text string.
* **Character Manipulation:** Demonstrates clean use of standard Java String methods and loops.
